# Daesupeee Bro
**Front-End** : 주여진, 정하진, 전선우

# Commit Message Rule
| 타입       | 설명                                                                 |
|------------|----------------------------------------------------------------------|
| **feat**   | 새로운 기능 추가                                                    |
| **fix**    | 버그 수정                                                           |
| **style**  | 코드 스타일 변경 (기능에 영향 없음, 예: 공백, 포맷팅 수정)           |
| **refactor** | 코드 리팩토링 (기능 변경 없음)                                      |
| **test**   | 테스트 코드 추가 또는 수정                                           |
| **chore**  | 기타 작업 (빌드 설정, 패키지 업데이트 등)                            |
| **docs**   | 문서 작업 (예: README.md 수정)
